This prototype provides a graphical interface to OpenFDA - Enforcement Reports, using OpenFDA publicly available API. This interface offers an insight to Recalls reported to FDA for Drugs, Devices and Foods. The following information is made available:

  • Number of recalls by Classification for each category - Drugs, Devices and Foods
  • Number of recalls by Status for each category - Drugs, Devices and Foods
  • Number of recalls by Voluntary/Mandated for each category - Drugs, Devices and Foods
  • List of recalled records for a selected data field, for each category - Drugs, Devices and Foods
